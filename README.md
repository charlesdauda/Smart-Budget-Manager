# Smart-Budget-Manager
SmartBudget is a personal finance web app that helps users manage their income, expenses, and balance easily.
It includes login and registration, transaction tracking, and charts to visualize your spending.

# Features
- User Authentication (Register, Login, Logout)
- Add, View, and Delete transactions
- Charts to show income vs expenses and category spending
- Dashboard Summary (Total Income, Total Expense, Balance, Total Transactions)
- Recent Transactions Table
- Clean and responsive user interface

# Built With
- PHP (Backend)
- MySQL (Database)
- HTML, CSS, JavaScript (Frontend)
- Chart.js (Data Visualization)
- Remix Icons (Icons & UI)

# Dashboard Includes
- Total Income
- Total Expenses
- Current Balance
- Transaction List
- Charts for Income vs Expenses & Category Spending
